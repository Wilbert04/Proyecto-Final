Mantemiento
-----------
Este es el registro mantenimiento con su detalle, antes de proceder a llenarlo es necesario tener registrado por lo menos un vehiculo y un cliente, luego que selecciona el cliente y su vehiculo procede a llenar la descripcion del mantenimiento que se le dará y elegir un servicio, por ultimo agrega la cantidad del articulo que se usó.
